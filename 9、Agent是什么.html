Agent 到底是什么
不是聊天机器人

By NOVA · 95% 说 Agent 的人 都说错了
01
PART · ONE
先打破 3 个常见误解
❌ 误解 · 95% 的人这么以为
这些都不是 Agent ⬇️
误区 1ChatGPT、豆包、文心一言——不是 Agent
误区 2Coze 上搭的聊天机器人——不是 Agent
误区 3能"回答问题"的 AI——只是大模型外壳
✅ 真相 · Agent 是另一种东西
Agent = 能自主完成任务的 AI 助理

你给它一个目标，它自己拆任务、找工具、执行、检查、犯错再试，直到搞定为止。

02
PART · TWO
一个类比讲透差别
🤓
大模型像实习生
你问什么
他答什么

问完就走
不会主动做事
VS
🤖
Agent 像全能助理
你给目标
他自己拆任务

自己找工具
自己执行交付
真实场景对比 · SAME QUESTION
👤 用户说 "帮我订一张明天去上海最便宜的机票"
❌ ChatGPT "建议你去
携程、飞猪比价

需要我帮你
看看怎么比吗？"
✅ AGENT "已查询 3 个平台
东航 SHA 580 元
已下单 + 选座
确认邮件已发 ✅"
03
PART · THREE
Agent 4 个核心能力
缺一个 = 不是 Agent，只是聊天框
🧠
能力 01
规划 Planning
把大目标拆成小步骤
能干啥 收到"订机票"——自己拆成"查航班 → 比价格 → 选座位 → 下单 → 发确认" 5 步。
例子 你说"帮我策划生日派对"——它自己想到要算预算、订场地、买蛋糕、发邀请。
没这个 只能等用户一步步问、一步步答。那叫聊天，不叫 Agent。
记忆 Memory
记住之前发生了什么
能干啥 短期：记住本次对话上下文
长期：记住你的偏好（爱坐靠窗、不要红眼航班）
例子 下次你说"再订一张"——它知道你还是要靠窗 + 不要红眼，不用重新问。
没这个 每次都从头问一遍。用户骂"你怎么又忘了"。
💾
能力 02
🔧
能力 03
工具调用 Tool Use
不光会说话，还会动手
能干啥 自己调 API、查数据库、发邮件、下订单、运行代码——把"答案"变成"行动"。
例子 订机票要调 3 个 API：航班查询 + 支付 + 邮件发送。Agent 自己串起来。
没这个 只能告诉你"建议去携程"，不能真的帮你订。这就是 ChatGPT 的天花板。
自我修正 Reflection
做错了能发现 + 重试
能干啥 订机票失败（库存不足）——自己换一个航班再试，不需要用户重新问。
例子 代码报错 → 看错误信息 → 改代码 → 再跑 → 还报错 → 换方案。一直试到成功。
没这个 一报错就摆烂："这个我搞不定"。这就是"假 Agent"的典型表现。
🔄
能力 04
📏 真假 AGENT 判断标准
下次有人跟你说
"我们做了一个 Agent"——
追问 "规划机制怎么设计的"
追问 "记忆怎么管理"
追问 "调用哪些工具"
追问 "失败怎么重试"

答不上来 → 他做的不是 Agent，是个套壳对话框

核心结论 · TAKEAWAY
所有不会自主调用工具的
"Agent" 都是假 Agent
